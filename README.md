# Роль 1

Для поручителя будет требоваться реализовать
следующий функционал:  
формирование записи (CRUD) по «сущности 10»;  
формирование записи (CRUD) по «сущности 11» с привязкой
записи по «сущности 10»;  
формирование записи (CRUD) по «сущности 12» с выбором из
списка «сущности 10» нескольких записей к ней;  
получение списка по «сущности 8» на основе выбранных записях
«сущности 10» в формате doc/xls;  
привязка к записи «сущности 12» записи «сущности 9»;  
получение отчета за период по «сущностям 11» и «сущностям 12»,
связанных между собой, в формате pdf на почту и вывод отчета на
форму;
